## ✨ Overview

ShopSphere is a frontend-only shopping interface designed to feel modern, interactive, and production-inspired.  
It focuses on delivering a polished user experience with smooth cart interactions, organized product browsing,
and a sleek glassmorphism-style UI.

This project is ideal for:
- frontend portfolio projects
- e-commerce UI demos
- JavaScript DOM practice
- responsive design showcases
- beginner-to-intermediate frontend learning

---

## 🚀 Features

- 📦 Responsive product grid
- 🔎 Live product search
- 🧩 Category filtering
- ↕️ Product sorting
- 👀 Product detail modal
- 🛒 Add-to-cart functionality
- 📂 Animated cart sidebar
- ❤️ Wishlist / saved items
- 💾 localStorage persistence
- 🖼️ Image hover zoom effect
- 💰 Subtotal, discount, tax, and total calculation
- 🚚 Free shipping progress tracker
- 🔔 Toast notifications
- 📱 Mobile-friendly responsive design

---

## 🎨 UI Highlights

- Modern glassmorphism-inspired interface
- Smooth hover interactions
- Clean card-based layout
- Responsive grid system
- Minimal yet professional shopping experience
